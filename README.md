ESP32 Wireless Embedded Communicator

Embedded communication system developed using ESP32, NRF24L01 and OLED displays.

Features
Bidirectional wireless communication
ACK protocol
OLED graphical interface
Menu system
Statistics screen
Transmission confirmation
Audible feedback
Modular firmware

Hardware

ESP32 DevKit V1 ×2
NRF24L01 ×2
OLED SSD1306 0.96" ×2
Push Button ×2
Active Buzzer ×2
